# Roduq Dark Cinematic — Premium SaaS Dark Mode

> Category: Roduq Premium / Developer Tools / Finance
> Tagline: "Premium, focused, immersive. Dla high-end SaaS / developer tools / finance."

## Visual Theme & Atmosphere

Confident, premium, technical. Jet black base + neon purple + cyan accents + subtle glows. Inter Display tight tracking, backdrop blur navigation, scroll-triggered fade-ins z restraint.

Best dla: high-end SaaS, developer tools, AI / ML platforms, finance / fintech, enterprise dashboards.

## Color Palette

### Primary
- **Background** (`#0A0A0A`): jet black — depth foundation
- **Surface raised** (`#141414`): cards, modals
- **Surface sunken** (`#000000`): below-fold sections
- **Surface elevated** (`#1F1F1F`): popovers, dropdowns
- **Foreground** (`#FAFAFA`): primary text — never pure white
- **Accent — Neon Purple** (`#7C5CFC`): primary brand
- **Cyan** (`#22D3EE`): secondary highlights, gradient stops

### Neutrals
- Muted: `#A1A1AA`
- Meta: `#71717A`
- Border: `rgba(255, 255, 255, 0.12)`
- Border soft: `rgba(255, 255, 255, 0.06)`

### Glows (signature)
- Primary glow: `radial-gradient(circle, rgba(124, 92, 252, 0.3) 0%, transparent 70%)`
- Accent glow: `radial-gradient(circle, rgba(34, 211, 238, 0.2) 0%, transparent 60%)`

## Typography

- **Display**: `Inter Display`, `Geist`, `Inter Tight` (weights 700-800). Tight tracking.
- **Body**: `Inter` (weight 400-500). Light weights hurt readability in dark — minimum 400.
- **Mono**: `Geist Mono`, `JetBrains Mono`

## Voice & Tone

- Confident, premium, technical
- Concise — every word earns its place
- Avoid hype words ("revolutionary", "game-changing") — let product speak
- Stats prominent ("99.99% uptime", "p99 < 50ms")

## Layout Principles

- Hero z subtle background gradient/mesh (radial glow centered)
- Sticky navigation z backdrop blur (`backdrop-filter: blur(12px)`)
- Cards z subtle border (1px rgba white 6-12%) + slight glow on hover
- Code blocks z dark syntax highlighting (Shiki: `github-dark-default` lub `dracula`)
- Scroll-triggered fade-ins + parallax restraint
- Charts/dashboards prominent (premium SaaS = data-heavy)
