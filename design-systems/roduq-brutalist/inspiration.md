# Roduq Brutalist — Inspiration

## Gold standard references

### [Are.na](https://are.na)
- Brutalist with restraint — not aggressive, just direct
- Black + white + one accent
- Borders define everything
- **Co kopiować**: layout rhythm, accent restraint

### [Stripe Press](https://press.stripe.com)
- Book product pages w brutalist confidence
- Strong typography hierarchy
- Color blocks dla section division
- **Co kopiować**: book/product page layout, blocked sections

### [Bloomberg Businessweek](https://www.bloomberg.com/businessweek)
- Magazine brutalism z color blocks
- ALL CAPS headlines selectively
- Strong border weights
- **Co kopiować**: section transitions, headline scale

### [Pentagram](https://www.pentagram.com)
- Studio site z work-as-statement
- Heavy use of asymmetric grids
- **Co kopiować**: work grid asymmetry

### [&Walsh](https://andwalsh.com)
- Bold typography statement hero
- Color block sections
- **Co kopiować**: color block hero, oversized headlines

## Polish-market context

- Brutalism rzadko stosowany w Polish business contexts (czytelnictwo gusto-tradycyjne)
- Best dla: gallerie / art / fashion / publishing / culture spaces / festivals
- Polish examples: galerie sztuki, magazyny kulturalne (np. dwutygodnik.com)

## Font references

- **Söhne** (Klim Type Foundry, premium) — modern Helvetica successor
- **Helvetica Now** (Linotype, premium) — refreshed Helvetica
- **Helvetica Neue** (system) — widely available fallback
- **Inter** (Google Fonts, free) — Helvetica-adjacent neutral fallback
- Polish charset: all support ą/ć/ę/ł/ń/ó/ś/ź/ż

## Anti-patterns

❌ **Soft colors** — brutalism = primary RGB (pure red / yellow / blue)
❌ **Rounded corners** — radius 0 throughout
❌ **Drop shadows** — no depth tricks, only borders
❌ **Subtle hover** — bold inversion (color flip)
❌ **Multiple accent colors w one screen** — restraint via discipline
✅ **Border 2-4px solid** — print-feel structure
✅ **Oversized display typography** (6rem-8rem hero common)
✅ **Asymmetric grids** — golden ratio
✅ **ALL CAPS sparingly** — selected emphasis
✅ **Color block sections** — yellow strip / red CTA / etc.
