# Roduq Tech Modern — Inspiration

## Gold standard references

### [Linear](https://linear.app)
- Mesh gradient hero (signature feature)
- Geist font — clean, modern, technical
- Section transitions z subtle parallax (restrained)
- Dark mode pierwsza-class
- **Co kopiować**: hero gradient mesh, typography hierarchy, button styles

### [Vercel](https://vercel.com)
- Bento grid features
- Real-time stats (deploy count counter)
- Heavy animations restraint
- **Co kopiować**: bento layout, large heading scales

### [Resend](https://resend.com)
- Perfect typography rhythm
- Code blocks inline (Shiki + syntax)
- Email previews jako product showcase
- **Co kopiować**: code block treatment

### [Cal.com](https://cal.com)
- Sections rhythm — alternating gray/white backgrounds
- Logos strip (social proof) prominent
- **Co kopiować**: section padding rhythm

### [Trigger.dev](https://trigger.dev)
- Developer-focused product positioning
- Heavy code examples inline
- **Co kopiować**: developer voice tone

## Anti-patterns

❌ **Mesh gradient overuse** — restrict do hero only (max 1 per page)
❌ **Multiple accent colors w standalone use** — magenta only w gradient pairings z indigo
❌ **Skill bars / badge graveyard** — looks dated, vibe wrong dla Tech Modern
❌ **Stock dev photos** (suit + laptop) — credibility killer
✅ **Product screenshots / dashboard mockups** — show real product
✅ **Code snippets inline** w copy
✅ **GitHub star count / npm downloads** widget gdy applicable
