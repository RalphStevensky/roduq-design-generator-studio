# Roduq Default — Inspiration & References

## Industry references studied

### [Linear marketing](https://linear.app)
- Clean structure, generous whitespace
- Inter typography w large hero scales
- Subtle scroll animations (fade-in, no overwhelming parallax)
- **Co kopiować**: section rhythm, type scale, button styles

### [Vercel docs / homepage](https://vercel.com)
- Typography rhythm exemplary
- Geist font (similar to Inter Tight) dla display
- Dark mode pierwsza-class
- **Co kopiować**: heading hierarchy, code block treatment

### [Stripe homepage](https://stripe.com)
- Color theory subtle but precise
- Gradient accents used sparingly
- Iconography precise and consistent
- **Co kopiować**: accent restraint, gradient direction

### [Tailwind UI / Refactoring UI](https://tailwindui.com)
- Production component patterns
- Spacing scale + color ramps reference
- **Co kopiować**: spacing rhythm, contrast ratios

### [shadcn/ui](https://ui.shadcn.com)
- Accessibility-first React components
- Composable patterns z radius-md defaults
- **Co kopiować**: form control patterns, focus states

## Anti-patterns observed

❌ **Heavy scroll animations** — Linear restraint > Cybertruck overkill
❌ **Multiple accents** — Roduq default = single `--accent` (cobalt)
❌ **Pure black/white** — vibrates, hard on eyes (use #0F0F0F + #FAFAFA)
❌ **Generic stock photos** — credibility killer dla B2B
✅ **Real product screenshots / illustrations** — Stripe / Linear approach
✅ **Specific metrics** — "5h saved/week" > "save time"
✅ **Polish character verification** — test "Łódź żółw pięć słów" w fontach
