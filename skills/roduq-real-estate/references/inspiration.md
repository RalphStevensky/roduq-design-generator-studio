## Layout patterns

### Hero
- **Search prominent**: large search bar w hero, filter by city/type/price
- **Split with listings**: text positioning left, hero listing tile right
- **Static hero z featured property**: single luxury listing as hero

### Listings grid
- **Grid 3-col**: dla 6+ properties (standard)
- **Carousel**: dla featured highlights (3-6)
- **List with map**: dla power users / search results pages

### Property card
- Hero photo (most attractive room — typically living room or facade)
- Price prominent (large, bold)
- Specs row: m² / rooms / floor / year
- Tags: For sale / For rent / Premium / Renovation needed
- Agent attribution

### Neighborhood pages
- Hero z map fragment + neighborhood description
- "Co znajdziesz" — local amenities
- Sample listings from that area
- Avg prices z chart
- Future infrastructure mentions (metro, parks)

## Anti-patterns

❌ **Bad photos** — phone snaps, no staging, dark/blurry
❌ **No price visible upfront** — "Cena na zapytanie" loses interest
❌ **Generic property descriptions** ("Beautiful apartment in great location")
❌ **Hidden fees** — VAT / commission / notary not mentioned upfront
❌ **No agent info on listing** — buyer wants to know who's selling
✅ **Professional photography** (HDR, staged, multiple angles)
✅ **Price + sqm + monthly costs visible**
✅ **Agent face + direct phone**
✅ **Neighborhood context** w listing (1-line about area)
✅ **Sold history** (builds trust dla agencies)
