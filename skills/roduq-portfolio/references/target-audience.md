# Portfolio — Target Audience Profiles

## Persona 1: Direct Client (Hiring Manager) (28-50)

- **Context**: Has a project, evaluating freelancers / studios
- **State**: Goal-directed, has timeline + budget
- **Decision speed**: Medium — 1-3 weeks, often parallel evaluation
- **Needs**:
  - Work similar to their project visible
  - Availability signal
  - Pricing clarity (or ballpark)
  - Direct contact
- **Trust signals**: relevant case studies, named clients, testimonials
- **Mobile usage**: medium (often desktop dla brief writing)

**Copy strategy**:
- Work grid hero (showcase first)
- About concise — "Specjalizuję się w X dla Y"
- Availability visible
- Direct email + Calendly link

## Persona 2: Agency / Studio (looking dla collaborators) (30-50)

- **Context**: Agency overflow work, contractor hire
- **Decision speed**: Fast (within 1 week)
- **Needs**:
  - Rate / hourly rate visible
  - Availability immediate
  - Quick onboarding signal
- **Trust signals**: past agency work, team-fit references
- **Mobile usage**: low

**Copy strategy**:
- "Available for contract work" prominent
- Hourly rate or day rate visible (gdy applicable)
- Tools list matching agency stack
- Past agency work mentioned

## Persona 3: Recruiter / Hiring Company (28-55)

- **Context**: Looking dla full-time hire
- **Decision speed**: 2-6 weeks (interview process)
- **Needs**:
  - Resume / CV downloadable
  - LinkedIn link
  - Skills matching JD
  - Location / remote willingness
- **Trust signals**: company logos, github contributions, articles

**Copy strategy**:
- "Open to full-time roles" section gdy applicable
- CV PDF downloadable
- LinkedIn prominent
- Location + remote/relocate willingness

## Persona 4: Press / Speaker Inquiry (30-55)

- **Context**: Podcast invite, conference speaking, interview
- **Decision speed**: Fast (matching availability)
- **Needs**:
  - Bio + headshot easy to grab
  - Speaking topics list
  - Previous talks / podcasts
  - Direct contact
- **Trust signals**: prior speaking, content output

**Copy strategy**:
- "Speaking" section gdy creator does this
- Bio (short + long versions)
- High-res headshot link
- Past talks list

## Persona 5: Curious Browser / Inspiration (any age)

- **Context**: Found via Awwwards / Twitter / referral
- **State**: Browsing, not buying
- **Decision speed**: N/A
- **Needs**: Aesthetic engagement, work depth
- **Outcome**: bookmark / follow / mention

**Copy strategy**:
- Memorable visual experience
- Easy social follow
- "Subscribe" optional dla newsletter

## Signal detection w brief

| Brief contains | Likely primary persona |
|---|---|
| "freelance contracts", "client projects" | Persona 1 (Direct Client) |
| "agency collaborators", "white-label" | Persona 2 (Agency) |
| "open to full-time", "looking for job" | Persona 3 (Recruiter) |
| "speaking", "podcast", "articles" | Persona 4 (Press) |
| "personal site", "blog focus" | Persona 5 (Browser) |

## Per-persona emphasis

| Element | Direct Client | Agency | Recruiter | Press | Browser |
|---|---|---|---|---|---|
| **Hero focus** | Work-led | Rate + availability | Role / location | Speaking topic | Personality |
| **About length** | Concise (specialization) | Tools fit | Full bio + skills | Bio (short/long) | Personal story |
| **CTAs** | Email + form | "Contract work" CTA | CV download + LinkedIn | Speaker form | Newsletter signup |
| **Pricing** | Range OR "request quote" | Day rate visible | N/A | N/A | N/A |
| **Photos** | Real (informal OK) | Professional | Professional headshot | High-res available | Personality-led |

## Polish-specific considerations

### Polish freelance market
- Common rates (2025): 100-300 PLN/h dla designers, 150-400 PLN/h dla developers
- Day rate often: 800-2400 PLN
- Some show rates publicly, some "request quote"
- VAT: most Polish freelancers JDG z 23% VAT — disclose

### Polish vs international clients
- Polish clients: PL + EN portfolio (international ambition signal)
- International clients: EN primary, PL secondary (or skip)
- Pricing currency: PLN dla Polish, EUR/USD dla international

### Skills credentials Polish convention
- Studia (degrees) — list optionally
- Certyfikaty — Google UX / Adobe / specific platforms
- Polski + język obcy poziom (C1 / B2)
- Open source contributions = strong signal

### Common Polish portfolio aesthetics
- **Minimalist** — popularny dla designers
- **Story-led** — fotografowie / illustrators
- **Tech-modern** — developers
- **Polish typography** — Inter / Roboto Mono / IBM Plex (free Polish-friendly)

### Polish portfolio platforms popular
- Behance — designers
- Dribbble — UI designers
- GitHub — developers
- LinkedIn — recruiters' first stop
- Instagram — photographers / illustrators
- Personal website still essential — owned channel
