# Agency Website — Content Patterns

> Copy structure dla `roduq-agency` skill. Work-led, confident, process-oriented voice.

## Voice principles

### Confident but evidenced
- ❌ "Jesteśmy najlepszą agencją brandingową w Polsce."
- ✅ "Rebrandowaliśmy 47 marek heritage. Wszystkie nadal istnieją 10 lat później."

### Work > words
- ❌ Long description of services bez concrete examples
- ✅ Service description = 1 sentence + 2-3 case study links

### Process visible
- Agency clients buy predictability. Show how the work happens.
- Universal 4-5 step pattern: Discovery → Strategy → Design → Build → Evolve

## Per-section copy templates

### Hero (statement-text)
- PL pattern: "{Outcome / positioning bez jargon}."
- "Brandy które rosną przez 10 lat."
- "Strony które klienci kochają. CFO też."
- "Identyfikacja wizualna z dystansem."

Subtitle: 1-2 zdania zoom-out — kim jesteście + dla kogo.
- "Studio brandingowe z Krakowa. Pracujemy z markami heritage w FMCG i retail."

### Services
**Per service template** (3-card grid):
```
{Service name — 1-2 słowa}
{1-sentence description — what's included}
{Outcome — what client gets}
{Optional: link to detail page}
```

Examples (PL):
- "Branding · Strategia, identyfikacja wizualna, guideline. · Spójność na każdym touchpoint."
- "Web design · UX research, design system, development. · Strony które mierzą sukces w przychodzie."
- "Content · Copy, photography, video. · Materiały które klienci faktycznie czytają."

### Case study card
```
[Client logo / hero image]
{Client name} | {Year}
{Project title} — "{1-sentence outcome}"
{2-4 metric tags}: "+47% organic traffic", "ROAS 4.2", "+23 NPS"
{Service tags}: Branding · Webdesign · CMS
[View case study →]
```

PL examples:
- "Acme Foods · 2025 · Rebrand kompletny — od logo do packagingu. +18% przychodu w 12 miesięcy. [Zobacz →]"

### Team
**Founders spotlight** (top of team page):
- Large photo + biography (2-3 paragraphs)
- Quote: "Wierzymy że..." (philosophy stmt)
- Linked: portfolio + LinkedIn + speaking engagements

**Team grid** (below founders):
- Photo + name + role + 1-line specialty
- Optional: "Recent project" link

### Process (universal 5-step)
1. **Discovery** — audit current state, stakeholder interviews
2. **Strategy** — positioning + messaging + voice
3. **Design** — identity + system + applications
4. **Build** — production + handoff + training
5. **Evolve** — ongoing review + optimization

PL: Odkrywanie · Strategia · Projekt · Wdrożenie · Ewolucja

### Contact
- **Brief form** preferred over generic "contact us"
- Fields:
  - Imię + nazwisko
  - Firma + rola
  - Budżet (range: <50k / 50-150k / 150-500k / 500k+ PLN)
  - Timeline (research / 3-6mc / 6-12mc / longer)
  - Co możemy dla Ciebie zrobić? (long text)
- **OR** Calendly embed dla discovery call

## Polish-specific

### Honorifics by audience
- B2B mid-market: neutral "Ty / wy" (less formal than enterprise) — most agencies operate here
- Enterprise (>500 employees): "Państwo"
- Startups / SMB: "Ty"

### Currency display
- "150 000 zł" (PLN default for Polish agency targeting Polish market)
- Or ranges: "50-150k PLN" (cleaner dla brief form)
- EUR/USD for international/CEE positioning

### Local trust signals
- Local press mentions (Marketing przy Kawie, Brief, Press)
- Polish award names (Effie Awards Polska, Klub Twórców Reklamy, KTR)
- Industry: IAB Polska membership

## Length constraints

| Section | PL chars max | EN chars max |
|---|---|---|
| Hero title | 70 | 60 |
| Hero subtitle | 200 | 180 |
| Service title | 25 | 20 |
| Service description | 120 | 100 |
| Case study title | 60 | 50 |
| Case study description | 150 | 130 |
| Team bio short | 100 | 80 |
| Team bio long | 400 | 350 |
| Process step | 50 | 40 |

## Anti-patterns

❌ **"Award-winning"** w hero — generic, used by everyone
❌ **Long generic philosophy paragraphs** ("We believe in...") — show through work
❌ **Hidden case study details** ("Get in touch dla pełne details") — kills trust
❌ **No process section** — mid-market clients need predictability
✅ **Specific metrics w case studies** — "+47% traffic" beats "significant growth"
✅ **Named team members** (no "Our experts" generics)
